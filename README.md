# applepay-southkorea-DataAnalysis
대한민국에서 애플페이 를 지원하는 가맹점에 대한 정보를 정리했습니다

## 목적
- 기존 애플페이를 지원하는 가맹점에 대한 지도는 API 문제 때문에 실용적이지 못했습니다. (API가 유료)
- 따라서, 애플페이를 지원하는 가맹점 리스트를 CSV로 정리하였습니다.
